<h2 style="text-align:center;">Fractal star</h2>
Fractal star implimented in pure C

### Pre-requirements
  Compiler: <a href="https://gcc.gnu.org/"><b>gcc</b></a> <br>
  Graphics library: <a href="https://raylib.com"><b>raylib</b></a>

### Compile
    chmod +x build.sh
    ./build.sh

### Run
    ./bin/fractal_star
